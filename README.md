## 📍 GhostySpawn

Lightweight spawn management plugin for all types of Minecraft servers.
Supports **Minecraft 1.21 – 1.21.11** (Paper & Spigot).

---

### ✨ Features

* Set and manage server spawn points
* Teleport players to spawn
* Custom join spawn teleport
* Simple & fast teleport system
* Supports `&` and HEX color codes
* No database required
* Low performance impact
* Fully customizable language files
* English & German support

---

### 🧾 Commands

| Command               | Description               | Permission               |
| -----------           | ------------------        |------------------        |
| `/addspawn`           | Set server spawn          | ghostyspawn.addspawn     |
| `/spawn`              | Teleport to spawn         | ghostyspawn.spawn        |
| `/ghostyspawn reload` | Reload plugin data        | ghostyspawn.reload       |
| /                     | Bypass the teleport delay | ghostyspawn.bypass-delay |

---

### ⚙️ Installation

1. Download the latest release
2. Place the `.jar` into `/plugins/`
3. Restart your server
4. Configure `config.yml`

---

### 🔄 Compatibility

* Paper
* Spigot
